## What "Limited" Means in Free Tier

**Enhanced Reading Features - Limited:**
- Basic annotations only (single color highlighting)
- 3 preset themes, no customization
- Maximum 3 collections
- Basic progress tracking (percentage only)

**AI-Powered Features - Limited:**
- 3 AI book summaries per month
- No chapter summaries
- No AI Q&A or smart search
- No translation

**Marketplace - Limited:**
- Browse and view book details only
- Cannot download or purchase books
- No author connections

**Analytics & Insights - Limited:**
- Basic stats only (books read, pages completed)
- No time tracking or goals
- No advanced analytics or data export

---

## Beta vs Alpha Features

**Pro Tier - Beta Features:**
- Access to features in beta testing (stable but new)
- Features that are 90% ready for general release
- May have minor bugs but core functionality works
- Examples: New reading modes, experimental AI features, beta integrations

**Plus Tier - Alpha Features:**
- Access to cutting-edge features in alpha testing (experimental)
- Earliest access to new innovations before anyone else
- Features may change significantly based on feedback
- Help shape the future of ReadEz
- Examples: Experimental UI concepts, bleeding-edge AI models, alpha partnerships
- Priority voting on feature roadmap

## Pricing Notes

**Pro Tier ($7.99/mo):**
- Positioned between Pocket Premium ($4.99) and Readwise ($7.99)
- Includes all stable features users need
- Beta access differentiates from competitors

**Plus Tier ($12.99/mo):**
- Premium tier for power users and early adopters
- Alpha access creates exclusivity
- Priority support justifies premium price
- Similar to Blinkist Premium ($11.99) but with more value
